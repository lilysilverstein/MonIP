MonomialIntegerPrograms :: monomialIdealsWithHilbertFunction

monomialIdealsWithHilbertFunction -- find all monomial ideals in a polynomial ring with a particular (partial or complete) Hilbert function

Synopsis

  • Usage:
    monomialIdealsWithHilbertFunction(L, R)
  • Inputs:
    • L, a list, {h(0), h(1), ..., h(d)}, the values of a valid Hilbert function of a graded ring for degrees 0...d.
    • R, a ring, a polynomial ring
  • Outputs:
    • a list, all ideals I of R that satisfy HF(R/I, i) = h(i) for all 0≤i≤d

Description

Macaulay’s Theorem characterizes all integer sequences which may be the Hilbert function of a graded ring. The package LexIdeals has several functions based on this theorem, including the function isHF which takes an integer sequence and returns true if it is a valid (partial) Hilbert function, and false otherwise.

When you call monomialIdealsWithHilbertFunction(L, R), first the function isHF L is called to make sure that the problem is feasible.

MonomialIntegerPrograms :: loadSCIPCodimAndDegree

loadSCIPCodimAndDegree -- change codim and degree to use the IP methods from this package

Synopsis

  • Usage:
    loadBuiltinCodimAndDegree()

Description

When the package is loaded, codim and degree are replaced with codimensionIP and degreeIP respectively for a MonomialIdeal. Any other functions that rely on codim, such as dim, are affected accordingly.

loadSCIPCodimAndDegree can be used to reload the the SCIP methods in the event that loadBuiltinCodimAndDegree was called.

i1 : R = QQ[a,b,c];
 
i2 : ScipPrintLevel = 1;
 
i3 : codim(monomialIdeal(a^2, b*a, c*b))
Codim files saved in directory: /tmp/M2-24912-0/0

o3 = 2
 
i4 : degree(monomialIdeal(a^2, b*a, c*b))
Codim files saved in directory: /tmp/M2-24912-0/1
Degree files saved in directory: /tmp/M2-24912-0/2

o4 = 3
 
i5 : loadBuiltinCodimAndDegree();
 
i6 : codim(monomialIdeal(a^2, b*a, c*b))

o6 = 2
 
i7 : degree(monomialIdeal(a^2, b*a, c*b))

o7 = 3
